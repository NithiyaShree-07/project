# 🏗️ BuildCost Pro — Construction & Repair Estimator

BuildCost Pro is a smart web-based estimator that helps users calculate **construction, painting, and repair costs** quickly and accurately.

It includes interactive dashboards, real-time calculations, and visual cost breakdowns.

---

## 🚀 Live Website
🔗 https://construction-painting-estimator.netlify.app/

---

## ✨ Features

### 🏠 Construction Estimator
- Estimate cost based on:
  - Area (sq ft)
  - Floors
  - Location (Metro / Tier cities)
  - Material & finishing quality
- Detailed **cost breakdown**
- Labour cost calculation
- Interactive **donut charts & graphs**

---

### 🎨 Painting & Repair Estimator
- Supports:
  - Painting only
  - Repair only
  - Combined work
- Choose:
  - Paint type (Emulsion, Enamel, Texture, etc.)
  - Brand (Asian, Berger, Dulux, etc.)
  - Finish (Matte, Gloss, etc.)
- Auto calculation of:
  - Paint quantity
  - Cost per litre
  - Labour cost
- Smart **paint info system**

---

### 📊 Dashboard & Analytics
- Total estimates tracking
- Lowest & highest cost tracking
- Session-based history
- Visual charts using Chart.js

---

### ⚙️ Smart Inputs
- Real-world parameters:
  - Labour wages
  - Working hours
  - Site conditions
- Dynamic UI with instant updates

---

## 🛠️ Technologies Used

## 🧠 How It Works

- Uses predefined cost models based on **Indian market rates (2024–25)**
- Calculates:
  - Material cost  
  - Labour cost  
  - Miscellaneous expenses  
- Displays:
  - Total estimate  
  - Category breakdown  
  - Visual analytics
